## Interessenter (uddybning)

### Interessent: Konsulenter
**Forventninger**
- Projektet fuldføres.
- Landmændene deltager.
- Behov for overblik over landmændenes status, aktiviter og resultater.
- Værdiskabelse.
- Hurtigere svartider til landmændene.
- Mindre arbejdspres.

**Bekymringer**
- Manglende engagement fra landmændene.
- Usikker koordinering mellem mange partnere.
- Systemets bliver ustabilt.

**Påvirkning**
- Positiv.

## Strategi for interessenthåndtering
Landmændene skal inddrages i projektet på en måde så de føler ejerskab. Samt grundig oplæring i systemet. Derudover bør man vise landmændene best case scenarier.

Koncern ledelsen frekvente rapporter og opdateringer. Indgå i branchenetværk for at følge udviklingen på området.

Konsulenter & Nature Connect Team skal have frekvente rapporter og opdateringer. Derudover skal relevante arbejdsgrupper etableres.

Myndigheder skal informeres om projektets fremdrift og resultater gennem officielle rapporter.

Lokalsamfundet skal informeres gennem lokale medier og events om projektets miljømæssige fordele.

## Krav

| Type                   | Krav                                                                 |
|------------------------|----------------------------------------------------------------------|
| Funktionelt krav       | Systemet skal kunne registerer 10 billeder per event.               |
| Funktionelt krav       | Hvor ofte bliver vidensdelingen brugt mellem landmændene.           |
| Funktionelt krav       | Hvor meget hurtigere bliver svartiderne.                             |
| Ikke-funktionelt krav  | Hvor ofte bliver Nature Connect brugt frem for Arlas konsulenter.   |
| Ikke-funktionelt krav  | Systemet skal være brugervenligt                                    |
| Ikke-funktionelt krav  | Systemet skal kunne tilgås af flere brugere                         |
| Ikke-funktionelt krav  | Systemet skal være pålideligt                                       |
| Forretningsmæssigt krav| Systemet skal kunne understøtte fremtidig udvidelse af programmet.  |
| Forretningsmæssigt krav| Skal holdes inden for budget                                        |

## KPI'er

| KPI                              | Målemetode                                             | Mål        | Målfrekvens        |
|----------------------------------|--------------------------------------------------------|------------|--------------------|
| 10 billeder per event            | Antal events * 10                                      | 100%       | Årlig              |
| Vidensdeling                     | Klik per event                                         | 20%        | Hver tredje måned  |
| Svartiderne                      | Svartid for konsulent mod svartid i systemet           | 5 min.     | Månedligt          |
| NC frem for konsulenter          | (Arlas konsulenter / Arla Nature Connect )             | 80%        | Hver tredje måned  |
| Brugervenlighed                  | Brugerundersøgelse hvor brugeren giver point 1-5       | gns >=3    | en gangs test      |
| Systemet tilgås af flere brugere | Stress test med antal personer                         | 100        | Årlig              |
| Systemet skal være pålideligt    | Stress test med antal personer                         |            | Årlig              |
| understøtter fremtidig udvidelse | Lavet ud fra principper som gør det muligt at udvide   | 3 måneder  | en gangs test      |
| Budget                           | Beregning                                              | =<2.000.000 kr | en gangs test   |
