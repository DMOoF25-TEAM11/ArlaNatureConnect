Detailed Description

Base ViewModel for list views. Provides the foundational properties and methods needed to manage and interact with a list of entities in the user interface, including loading entities and handling common operations.

Template Parameters
TReposThe type of the repository used to access the entities. Must implement IRepository<T>.
TEntityThe type of the entities being managed. Must be a reference type.

Derive concrete list view models from this class to obtain a repository reference, shared status and messaging services, and a standard asynchronous LoadAsync(Guid) pattern.
