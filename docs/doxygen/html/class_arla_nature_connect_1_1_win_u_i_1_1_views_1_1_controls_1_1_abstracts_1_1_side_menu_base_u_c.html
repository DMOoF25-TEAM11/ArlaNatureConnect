Detailed Description

Generic base UserControl for side-menu user controls. Handles subscribing/unsubscribing to DataContext PropertyChanged and optionally calling a parameterless InitializeAsync() method on the view-model if present. Derived side-menu controls can override hooks to react to VM set / property changes.
