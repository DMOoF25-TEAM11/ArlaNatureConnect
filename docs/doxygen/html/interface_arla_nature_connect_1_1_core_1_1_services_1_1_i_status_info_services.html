Detailed Description

Provides application-wide status information and control for UI components and services.

This interface abstracts global status such as whether the application is currently performing a long-running operation and whether a database connection is available. Consumers (views, view-models, and services) use this interface to observe changes via StatusInfoChanged and to enter a loading scope via BeginLoadingOrSaving. The scoped loader returned by BeginLoadingOrSaving ensures IsLoadingOrSaving is set while an operation runs and automatically resets when the scope is disposed. Abstracting this behavior enables easier testing and decouples UI concerns from concrete implementations.

IDisposable ArlaNatureConnect.Core.Services.IStatusInfoServices.BeginLoadingOrSaving ( )

Begins a loading scope and returns an IDisposable that ends the scope when disposed.

Returns
An IDisposable which, when disposed, will end the loading scope (typically by setting IsLoadingOrSaving to false).

Use this method to ensure the IsLoadingOrSaving flag is set for the duration of an operation. Typical usage:

using (status.BeginLoadingOrSaving()) { await DoWorkAsync(); }
