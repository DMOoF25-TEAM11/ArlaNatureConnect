An optional name of the current entity or context the messages relate to.

Consumers can set this to provide additional context for displayed messages (for example, the identifier of the record currently being edited). Implementations should raise INotifyPropertyChanged.PropertyChanged when the value changes.

The collection is intended to be consumed by the UI (for example, rendered in a status panel). Implementations may return an immutable snapshot or a live view. Consumers should not attempt to modify the returned collection directly.
