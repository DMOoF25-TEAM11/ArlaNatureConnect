async Task ArlaNatureConnect.WinUI.ViewModels.Controls.SideMenu.FarmerPageSideMenuUCViewModel.InitializeAsync ( )
inline

Performs asynchronous initialization for the view-model.

Returns
A task that completes when the available persons for the farmer role have been loaded.

This method deliberately delegates to the shared SideMenuViewModelBase.LoadAvailablePersonsAsync(string) implementation so role-specific view-models only need to specify the role name. Keeping this method small makes it easier to call from the view's lifecycle code (for example when the view is loaded) without duplicating repository logic.
