◆ LoadAsync()

async Task ArlaNatureConnect.WinUI.ViewModels.Abstracts.CRUDViewModelBase< TRepos, TEntity >.LoadAsync ( Guid id)
inline

Loads an entity by id and prepares the view-model for edit mode if the entity exists. This method hides the base implementation to augment loading with form hooks and edit-mode state.

Parameters
idIdentifier of the entity to load.
Returns
A task that represents the asynchronous load operation.

The implementation uses IStatusInfoServices.BeginLoadingOrSaving to report loading state and calls OnLoadFormAsync(TEntity) when an entity is found. Exceptions are swallowed intentionally to keep UI flows predictable; derived implementations should surface errors via IAppMessageService when appropriate.
