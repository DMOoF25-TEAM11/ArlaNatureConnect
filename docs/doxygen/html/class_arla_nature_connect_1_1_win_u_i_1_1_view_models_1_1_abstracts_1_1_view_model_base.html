Detailed Description

Base class for all ViewModels providing common functionality. Implements INotifyPropertyChanged to support property-change notifications required by data-binding frameworks (e.g. WinUI, WPF, UWP).

Derive concrete ViewModel classes from this abstract class. Use OnPropertyChanged(string?) within property setters to notify the UI of updates. Do not attempt to instantiate this abstract class directly.

Parameters
nameThe name of the property that changed. This parameter is optional; when omitted, the compiler will supply the caller member name due to the CallerMemberNameAttribute.

Use this method in property setters to notify listeners of changes, for example:

private string? _name;
public string? Name
{
get => _name;
set
{
if (value == _name) return;
_name = value;
OnPropertyChanged(); // CallerMemberName provides "Name"
}
}
