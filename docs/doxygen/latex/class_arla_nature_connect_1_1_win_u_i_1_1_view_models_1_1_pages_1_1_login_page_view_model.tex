\doxysubsection{Detailed Description}
View\+Model for \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_views_1_1_pages_1_1_login_page}{Login\+Page} -\/ handles role selection and navigation to different dashboards. 

Responsibilities\+:
\begin{DoxyItemize}
\item Receives role selection from UI (Farmer, Consultant, Arla Employee)
\item Creates \doxylink{class_arla_nature_connect_1_1_domain_1_1_entities_1_1_role}{Role} object based on the selection
\item Navigates to the correct Page (\doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_views_1_1_pages_1_1_farmer_page}{Farmer\+Page}, \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_views_1_1_pages_1_1_consultant_page}{Consultant\+Page}, or \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_views_1_1_pages_1_1_arla_employee_page}{Arla\+Employee\+Page})
\item Passes the \doxylink{class_arla_nature_connect_1_1_domain_1_1_entities_1_1_role}{Role} object to the next Page via \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_services_1_1_navigation_handler}{Navigation\+Handler}
\end{DoxyItemize}

Usage\+: This View\+Model is used by Login\+Page.\+xaml to handle the user\textquotesingle{}s role selection. When the user clicks a role button, the role name is sent as a parameter to Select\+Role\+Command, which then navigates to the correct Page with the selected \doxylink{class_arla_nature_connect_1_1_domain_1_1_entities_1_1_role}{Role} object.
