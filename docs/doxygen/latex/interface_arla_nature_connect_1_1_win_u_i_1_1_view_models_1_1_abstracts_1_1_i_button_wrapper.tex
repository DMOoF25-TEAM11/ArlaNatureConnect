\doxysubsection{Detailed Description}
Represents a minimal wrapper for a button-\/like element used by view models. 

This abstraction allows view models to interact with button controls or button-\/like view elements without referencing UI framework types directly. It is useful for decoupling, unit testing, and providing a uniform contract for passing command parameters or arbitrary metadata (Tag) from views to view models.
