EF \doxylink{namespace_arla_nature_connect_1_1_core}{Core} configuration for \doxylink{class_arla_nature_connect_1_1_domain_1_1_entities_1_1_nature_check_case}{Nature\+Check\+Case} entity. 

Configures type conversions between the domain model and database schema\+: 
\begin{DoxyItemize}
\item Status enum is stored as NVARCHAR(50) with CHECK constraint (\textquotesingle{}Assigned\textquotesingle{}, \textquotesingle{}In\+Progress\textquotesingle{}, \textquotesingle{}Completed\textquotesingle{}, \textquotesingle{}Cancelled\textquotesingle{}) 
\item Date\+Time\+Offset properties (Created\+At, Assigned\+At) are stored as DATETIME2 in the database 
\end{DoxyItemize}The configuration ensures proper conversion when reading from and writing to the database, including handling of nullable values and default status assignment for invalid or missing values.
