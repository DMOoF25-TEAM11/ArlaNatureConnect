Inherits \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_view_models_1_1_abstracts_1_1_list_view_model_base-2-g}{List\+View\+Model\+Base$<$\+IPerson\+Repository, Person$>$} specialized for the \doxylink{class_arla_nature_connect_1_1_domain_1_1_entities_1_1_person}{Person} entity. How to use\+:
\begin{DoxyItemize}
\item Construct or resolve a concrete side-\/menu view-\/model and set it as the Data\+Context for the side-\/menu User\+Control.
\item Call \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_view_models_1_1_abstracts_1_1_side_menu_view_model_base_a0d89c5470d8218b71201ad857c4f4e48}{Load\+Available\+Persons\+Async(string)} with the desired role to populate \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_view_models_1_1_abstracts_1_1_side_menu_view_model_base_a70a3768d117649f7471df105f31869e8}{Available\+Persons}.
\item Bind \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_view_models_1_1_abstracts_1_1_side_menu_view_model_base_a7d7843cf83627560f72675b0435cf3ad}{Selected\+Person} to the view\textquotesingle{}s selection control and hook up \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_view_models_1_1_abstracts_1_1_side_menu_view_model_base_a63c500cf8930e0624a5ae0f801a8ff31}{Navigation\+Command} / \doxylink{class_arla_nature_connect_1_1_win_u_i_1_1_view_models_1_1_abstracts_1_1_side_menu_view_model_base_a72316d59aed2c71c2df18762c7536add}{Logout\+Command} to buttons.

Why we have it\+:

\end{DoxyItemize}
