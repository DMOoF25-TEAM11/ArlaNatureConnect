\doxysubsection{Detailed Description}
Provides application-\/wide status information and control for UI components and services. 

This interface abstracts global status such as whether the application is currently performing a long-\/running operation and whether a database connection is available. Consumers (views, view-\/models, and services) use this interface to observe changes via \doxylink{interface_arla_nature_connect_1_1_core_1_1_services_1_1_i_status_info_services_a5fd6e3c1f7aca8fb9a37359f83333346}{Status\+Info\+Changed} and to enter a loading scope via \doxylink{interface_arla_nature_connect_1_1_core_1_1_services_1_1_i_status_info_services_a6e098a14ecc16a31f815606c84472c26}{Begin\+Loading\+Or\+Saving}. The scoped loader returned by \doxylink{interface_arla_nature_connect_1_1_core_1_1_services_1_1_i_status_info_services_a6e098a14ecc16a31f815606c84472c26}{Begin\+Loading\+Or\+Saving} ensures \doxylink{interface_arla_nature_connect_1_1_core_1_1_services_1_1_i_status_info_services_a481e1640fef9c62cae1a373da5a953e0}{Is\+Loading\+Or\+Saving} is set while an operation runs and automatically resets when the scope is disposed. Abstracting this behavior enables easier testing and decouples UI concerns from concrete implementations.
