\doxysubsection{Detailed Description}
Base page that hosts a navigation-\/enabled view-\/model and wires common navigation lifecycle behavior. 

Why this class exists\+:
\begin{DoxyItemize}
\item Centralizes the common setup performed when navigating to role-\/based pages\+: validating the view-\/model, invoking its asynchronous initializer with the current \doxylink{class_arla_nature_connect_1_1_domain_1_1_entities_1_1_role}{Role}, selecting the default navigation item and attaching the side menu to the main window.
\item Keeps page classes small by moving shared navigation/side-\/menu wiring to a single abstract base.
\item Ensures consistent behavior (initialization, default selection, side-\/menu attach/restore) across all pages that present role-\/scoped dashboards or content. 
\end{DoxyItemize}
