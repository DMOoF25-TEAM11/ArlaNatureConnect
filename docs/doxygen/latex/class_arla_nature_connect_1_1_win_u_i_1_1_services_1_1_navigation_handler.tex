\doxysubsection{Detailed Description}
Concrete frame-\/based navigation handler used by the \doxylink{namespace_arla_nature_connect_1_1_win_u_i}{Win\+UI} application. This class implements \doxylink{interface_arla_nature_connect_1_1_win_u_i_1_1_services_1_1_i_navigation_handler}{INavigation\+Handler} and provides the actual navigation behavior using a Frame instance. 

Why we have it\+: This concrete implementation decouples navigation consumers (view-\/models, services) from platform-\/specific navigation logic while still providing a simple API. By depending on \doxylink{interface_arla_nature_connect_1_1_win_u_i_1_1_services_1_1_i_navigation_handler}{INavigation\+Handler}, callers become testable and DI-\/friendly.

\item Initialize the implementation once with the application\textquotesingle{}s main Frame (typically from the main window).
