KPI

## Requirements

| ID    | Type                      | Requirement                                                                 |
|-------|---------------------------|------------------------------------------------------------------------------|
| R001  | Functional requirement    | The system must be able to register 10 images per event.                     |
| R002  | Functional requirement    | Frequency of knowledge sharing usage among farmers.                         |
| R003  | Non-functional requirement| How often Nature Connect is used instead of Arla's consultants.             |
| R004  | Non-functional requirement| The system must be user-friendly.                                            |
| R005  | Non-functional requirement| The system must be accessible by multiple users.                             |
| R006  | Non-functional requirement| The system must be reliable.                                                  |
| R007  | Non-functional requirement| (No content provided)                                                         |
| R008  | Business requirement      | The system must support future expansion of the application.                 |
| R009  | Business requirement      | Must be kept within budget.                                                   |
| R010  | Non-functional requirement| CRUD operations must be performed within an acceptable time frame.          |

## Functional KPI measurements

| ID    | KPI                         | Measurement method           | Target | Frequency            |
|-------|-----------------------------|------------------------------|--------|----------------------|
| K001  | 10 images per event         | Number of events * 10        | 100%   | Annual               |
| K002  | Knowledge sharing           | Clicks per event             | 20%    | Every three months   |

## Non-functional KPI measurements

| ID    | KPI                         | Measurement method                                               | Target             | Frequency            |
|-------|-----------------------------|------------------------------------------------------------------|--------------------|----------------------|
| K003  | Response times              | Consultant response time vs. system response time                 | 5 min.             | Monthly              |
| K004  | Nature Connect vs consultants| (Arla's consultants / Arla Nature Connect)                       | 80%                | Every three months   |
| K005  | Usability                   | User survey where users rate 1-5                                  | avg >= 3           | One-time test        |
| K006  | System accessibility        | Stress test with number of users                                 | 100 users          | Annual               |
| K007  | System reliability          | Stress test with number of users                                 | (not specified)    | Annual               |
| K010  | CRUD operations time        | Average time for CRUD operations                                 | < 500 milliseconds | Monthly              |

|-------|-----------------------------|------------------------------------------------------------------|--------------------|----------------------|
