# Operation Contracts – UC002B.4: Create Farm (Inline)

---

## **Operation Contract – UC002B.4: Create Farm**

| Item | Description |
|------|-------------|
| **Operation** | Create a new farm in the system with its address and owner information, or reuse an existing farmer. |
| **Preconditions** | • All required farm information is provided.<br>• CVR number is unique in the system.<br>• If owner email exists, person must have Farmer role. |
| **Postconditions** | • New farm is created.<br>• Address is created and linked.<br>• Owner is created or reused (if Farmer role).<br>• Farm is available for assignment. |
| **Input** | • Farm name, CVR, address, owner information |
| **Output** | • Created Farm entity |
| **Exceptions** | • `InvalidOperationException` if CVR already exists<br>• `InvalidOperationException` if owner email exists but not Farmer role |
