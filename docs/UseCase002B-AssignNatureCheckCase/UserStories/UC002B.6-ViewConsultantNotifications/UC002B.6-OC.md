# Operation Contracts – UC002B.6: View Consultant Notifications

---

## **Operation Contract – UC002B.6: Load Consultant Notifications**

| Item | Description |
|------|-------------|
| **Operation** | Load all Nature Check Case assignments for a specific consultant. |
| **Preconditions** | • The consultant exists in the system.<br>• The consultant is authenticated. |
| **Postconditions** | • All assigned cases are loaded.<br>• Cases are sorted by assignment date (newest first).<br>• ConsultantNotificationDto objects are generated from case data (not stored in database). |
| **Input** | • Consultant ID |
| **Output** | • List of ConsultantNotificationDto |
| **Exceptions** | • `NotFoundException` if consultant does not exist |
