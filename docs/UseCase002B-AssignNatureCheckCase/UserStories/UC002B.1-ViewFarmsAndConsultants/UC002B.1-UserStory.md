# User Story – UC002B.1: View Farms and Consultants for Assignment

**As an** Arla employee,  
**I want** to view a list of all farms and consultants in the system,  
**so that** I can select a farm and consultant when assigning Nature Check Cases.

---

## Acceptance Criteria

1. **Given** I am an authenticated Arla employee  
   **When** I open the Nature Check Cases module  
   **Then** I see a list of all farms in the system

2. **Given** I am viewing the farm list  
   **When** the farms are displayed  
   **Then** each farm shows its assignment status ("Tilføjet" / "Ikke tilføjet")

3. **Given** I am viewing the farm list  
   **When** a farm has an active Nature Check Case  
   **Then** the farm displays the assigned consultant's name and priority level

4. **Given** I am viewing the consultant list  
   **When** consultants are displayed  
   **Then** only persons with the "Consultant" role are shown

5. **Given** I am viewing the consultant list  
   **When** consultants are displayed  
   **Then** consultants are sorted alphabetically by name

6. **Given** I am viewing the farm list  
   **When** farms are displayed  
   **Then** farms are sorted alphabetically by name

---

## Notes

- This is a read-only operation (no data modification)
- The view aggregates data from multiple sources (Farms, Persons, Addresses, NatureCheckCases)
- Assignment status is determined by checking for active Nature Check Cases
- Priority badges are displayed for farms with active cases
