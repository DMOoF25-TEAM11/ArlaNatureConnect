# Operation Contracts – UC002B.5: Delete Farm

---

## **Operation Contract – UC002B.5: Delete Farm**

| Item | Description |
|------|-------------|
| **Operation** | Remove a farm from the system. |
| **Preconditions** | • The farm exists in the system.<br>• The farm has no active Nature Check Cases. |
| **Postconditions** | • The farm is removed from the system.<br>• The farm no longer appears in the farm list. |
| **Input** | • Farm ID |
| **Output** | • Success confirmation |
| **Exceptions** | • `InvalidOperationException` if farm has active cases |
